javaScript之创建对象的两种方式

javaScript
  • 工厂模式 寄生模式
  • 工厂模式,寄生模式

    工厂模式

  • 在判断this是谁的时候,根本不需要考虑函数内部的功能,只需要看函数时怎么被调用的即可
  • 工厂模式的调用模式,是函数调用模式,所以this指向window

寄生模式

  • 寄生模式的调用模式是构造函数调用模式,所有this指向的就是new创建出来的对象
