AngularJS之基本概念

Angular JS
  • Angular JS
  • Angular JS的基本概念,控制器及应用基本步骤简介

    Angular JS MVC架构

  • 由Google公司开发维护的前端mvc框架
  • jQuery准确来是只是一个类库, 以DOM作为驱动(核心)
  • AngularJS则是一个框架,以数据和逻辑作为驱动(核心)
  • AngularJS最为核心的特性为: 模块化,双向数据绑定,语义化,标签,依赖注入等
  • 与之类似的框架还有: backBone, KnokoutJS Vue React
  • Model —— 模型
    • 是一个低水平负责维护数据的模式
    • 负责管理应用程序的数据
    • 响应从视图请求,同时也从控制器响应指令到自我更新
  • View —– 视图
    • 这是负责显示所有或一部分的数据到用户
    • 在一个特定的合适的呈现数据,由控制器决定呈现数据触发
  • Controller —– 控制器
    • 软件代码控制模型和视图之间的相互作用
    • 负责响应于用户输入并执行交互数据模型对象
    • 控制器接收输入,并验证输入,然后执行修改数据模型的业务操作的状态
  • Angular不依赖于其他库

Angular JS 控制器

  • 主要依赖于控制器来控制数据流在应用程序中
  • 控制器使用ng-controller指令定义
  • 控制器是一个包含属性/属性和函数的javaScript对象
  • 每个控制器接收$scope作为参数引用到应用程序/模块,由控制器来控制

创建Angular JS应用基本步骤

  1. 加载框架
    • 使用<script src=""> </script>标签来添加
  2. 使用np-app指令定义Angular JS应用
    • <div ng-app ="" > </div>
  3. 用ng-model指令定义的模块名称
    • <p> Enter you Name: <input type="text" ng-model="name"> </p>
  4. 用ng-bing指令将上述模型中的值绑定定义
    -<p>Hello <span ng-bind="name"></span>!</p>

流程:

  1. 页面加载完毕,angular开始对页面处理,寻找ng-app指令,找到这个指令之后,就确认APP容器元素了,此时开始对该容器元素内部的元素进行处理
  2. 此时遍历到input元素,这个元素上有ng-model指令,此时angular将该元素的内容绑定在作用域中的msg变量中
  3. 遍历到P元素,此时p元素进行了一个插值操作,angular会将作用域中的变量的值插入到 P元素内
  4. 遍历完成,将angular应用程序启动,我们就可以体验双向绑定的功能
