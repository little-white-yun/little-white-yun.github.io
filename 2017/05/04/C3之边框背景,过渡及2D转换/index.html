CSS3之边框背景,过渡及2D转换

CSS3
  • 边框背景,过渡及2D转换
  • C3新标签,box-shadow,border-image,background-size等 ####边框阴影 box-shadow
  • 四个参数:
    box-shadow : x轴方向的偏移 , y轴方向的偏移 , 阴影模糊度 , 颜色 ;
    例如: box-shadow : 3px 3px 5px #ccc // 边框向x轴偏移3px,y轴偏移3px,模糊度为5px,颜色为灰色
  • 五个参数 :
    box-shadow : x轴方向的偏移 , y轴方向的偏移 , 阴影模糊度 , 延伸阴影的大小 , 颜色 ;
  • 六个参数 :
    box-shadow : 内阴影( inset ) , x轴方向的偏移 , y轴方向的偏移 , 阴影模糊度 , 延伸阴影的大小 , 颜色 ;
    注: 若是需要加多个阴影,则可以直接逗号隔开即可;

####边框背景图片 border-image

  • 资源地址: border-image-source : url( “地址” );
  • 裁剪的尺寸: border-image-slice : 数值 数值 数值 数值; ( 没有单位 ,同样遵循 上 右 下 左 )
  • 边框的宽度: border-image-width : 像素值;
  • 平铺方式: border-image-repeat : 参数;
    • stretch —– 指定用拉伸方式来填充边框背景
    • repeat —– 从中心平铺方式来填充,图片碰到边界时,若是超过则被截断
    • round —– 平铺方式,图片会根据边框尺寸自适应调整图片大小

背景大小 background-size

  • 参数一: 数值
    • background-size : 数值 数值; 指定背景图片的大小
  • 参数二: 百分比
    • background-size ：百分比
  • 参数三: auto
    • 图片的真实大小
  • 参数四 : cover
    • 图片等比缩放到盒子内( 以盒子的最大边为比例缩放 ), 超出盒子部分隐藏,
  • 参数五：contain
    • 图片依据最窄边为比例等比缩放,可能会存在图片不及盒子大的情况

####背景显示 background-origin

  • 参数一 : padding - box
    • background-origin : padding - box; // 从padding区域开始显示背景。
  • 参数二 : content - box
    • background-origin :　content-box; // 从内容区域开始显示背景
  • 参数三 : border - box
    • background-origin : border-box ; //从边框区域开始显示背景

渐变色

linear - gradient 线性渐变
  • background - image : linear-gradient ( to 渐变的方向 , 起始的颜色 , 结束的颜色 )
  • 渐变的方向:
    • 方向值: top bottom right left
    • 角度: deg 如果是正的角度 , 顺时针; 如果是负的角度 , 逆时针; ( 0deg 朝上的方向 )
      注:
  • 可以通过百分比设置渐变长度
  • 依据高的长度,通过百分比先写出一个小格的样式,由于背景平铺可产生图2的效果
radial - gradient 径向渐变
  • background - image : radial - gradient ( 辐射半径 , 中心点的位置 , 起始颜色 , 结束颜色 )
  • 中心点的位置是以盒子自身为参照
  • 例如: background - image : radial - gradient( 150px at center center , yellow , green );

过渡属性 transition

  • 如果加上过渡属性, 具备了做补间动画的能力
  • 只要属性状态发生改变,就能触发过渡,产生动画
  • 第一个参数: all 所有属性都具备了过渡的能力
  • 第二个参数: 过渡的时间,单位: 秒 s
  • transition : all 3s;
  • 当需要为属性单独设置过渡事件和过渡样式的时候,可通过逗号间隔,在逗号后面直接书写

  • 拆分写法:

    • transition-property : all ; 过渡的属性
    • transition - duration : 4s; 过渡的时长
    • transition - timing- function : 参数; 过渡的速度
    1. linear 匀速
    2. ease 先快后慢 默认的速度
    3. ease - in 先慢后快
    4. ease- out 先快后慢
    5. 坐标变换( 贝塞尔曲线 )
    • transition - delay : 2s; 过渡的延时

  • 组合写法:

    • transition : 属性 时间 过渡的速度 过渡的延时时间 ;
    • 例如:

transition: all 5s ease-in 1s;
// 等价于
transition - property:all; // 全部元素过渡
transition - duration ：4s; // 持续时间4秒
transition - timing - function : ease-in; //过渡速度 先快后慢
transition - delay : 2s; // 过渡延时2秒

2D转换 transform

转换的中心点 transform - origin

  • 中心点是一个默认的转换原点

缩放 scale ( )

  • 一个参数
    • 代表整体缩放的倍数,比1大 —– 放大; 比1小 —– 缩小
  • 两个参数
    • transform : scale( x 轴方向的缩放 , y轴方向的缩放 );

旋转 rotate( )

  • 一个参数
  • transform : rotate( 旋转的角度 ) 单位deg;　
  • 如果是正角度,那么是顺时针旋转
  • 如果是负角度,那么是逆时针旋转

移动 translate( )

  • 两个参数
  • transform ：translate( x轴方向的位移 , y轴方向的位移 );
  • 注: 元素的位移,是基于元素的最后位置进行位移
  • 与旋转在一起使用,则要注意先写坐标,再旋转. 如果先旋转,那么坐标轴也发生旋转

倾斜 skew( )

  • 两个参数
  • transform : skew( x轴方向倾斜的角度, y轴方向倾斜的角度 )

    注意;
    x轴方向的倾斜, 如果角度是正的,则是逆时针,反之,角度是负的,就是顺时针
    y轴方向的倾斜,如果角度是正的,则是顺时针,反之,如果角度是负的,则是逆时针
    设置倾斜度的时候,一般是只设置一个值,另一个为0deg , 不建议同时设置两个倾斜度
