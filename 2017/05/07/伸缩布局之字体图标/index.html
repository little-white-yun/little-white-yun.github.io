伸缩布局类之字体/图标

布局综合类
  • 字体 图标
  • 伸缩布局,web字体的使用介绍
  • 父容器
    1. display : flex 声明伸缩容器
    2. flex-wrap : nowrap | wrap 控制是否可以换行
    3. flex-direction : row | column 排列方向

web字体

  • 说明:
    • 使用某一个字体的时候,会出现不支持特殊字体,从而使用电脑默认字体显示
      • 使用的字体,其实是会有约束文件( 指定某一个字符对应的图像) 称为: 字体文件
      • 如果系统里面没有这种文件约束, 使用的是web 是需要从服务端索取 ,索取字体文件,解决客户系统当中不支持的字体

使用方法:

  1. 下载字体文件
  2. 字体文件格式: eot avg woff woff2 , 用来解决不同浏览器兼容的
  3. 定义第三方字体 font - face指令: 告诉web浏览器字体名称和下载地址
    1. 使用字体
    • 字体定义:
      @font - face {
             font-family : " ";
             src : url ( " ");
      }
  • 规范写法:

@font-face {
font-family: 名称;
src:
url(".eot?#iefix") format("embedded- opentype");
url(".woff") format("woff");
url(".ttf") fortmat("truetype");
url(".svg") fortmat("svg");
}

  • font-family 为字体设定名称
  • src : 告诉浏览器在服务器什么位置寻找字体文件
  • font - family属性: 指定电脑中已安装的字体
  • 使用字体
    • 直接调用自定义字体的名称. font-family: “ “;

web字体图标 iconfont

  • 字体约束文件里面有自己的编码格式
  • 在HTML当中要使用一段字符表示一个图标是个字体文件的编码保持一致
  • 写在content 当中,字符编码保存一致,还需要转移符
  • 单个字体图标使用方法:
    1. 依据字体风格样式定义
    2. span::before{
      font-family:名字;
      content:"\编码";
      }
  • 总结:
  1. 矢量 —– 不管变大变小 , 都不失真
  2. 纯色 —– 只能代表一种颜色的图标,当做文字处理, 仅可改变 —– 阴影 , 颜色 , 大小等
  3. 加载 —– 集成在一个文件内,可提高加载速度
  • 字体图标库使用方法:
  1. 引入下载图标库的CSS文件 link
  2. 在要引用的标签中 直接添加类名引用( 字体名 图标名 )
  • 制作自定义的字体图标
  1. UI制作psd设计稿
  2. 抠出一个图标 jpg png ( 无法直接制作成字体图标 )
  3. 需要转换成另一种格式图片 svg 格式
    1. SVG图片,在线制作字体图标( http:// icomoon.io/ )
  4. 转换成字体文件
  5. 使用方法同字体图标使用一样
