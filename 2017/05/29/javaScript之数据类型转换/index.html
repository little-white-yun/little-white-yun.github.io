javaScript之数据类型转换 in的用法等

javaScript
  • 数据类型转换 in的用法 delete,对象的动态特性,异常等介绍
  • 数据类型转换,in的用法,delete
  1. 先调用这个引用值的valueOf方法,获取返回值之后参与运算,如果能运算,那么直接得出结果
  2. 如果不能运算,继续调用该引用类型数据的toString方法,获取返回值参与运算,这个肯定是可以得出来结果的!

in 的用法

for in

  • 遍历对象的属性
  • 对象的属性名都是string类型的

属性名 in 对象

  • 判断对象能否访问某个属性
  1. 名字(); 函数的直接调用，且立即直接调用
    div.onclick = 名字;
    注：
    1. 事件调用 —— 可以调用有名函数，也可以调用匿名函数
    2. 直接调用 —— 只能调用有名函数
      例：div.onclick = function( ){ .... }
    3. 匿名调用

delete

  • 删除没有使用var声明的变量
    • 语法:delete 变量名

  • 删除对象的属性

    • delete 对象名. 属性名
    • delete 对象名[属性名]

  • delete 关键字是有返回值的.表示删除的是否成功

    • 但是不可靠,一般不用

  • 访问一个不存在的变量.会报错! XX is not defined

  • 访问一个对象不存在的属性.不会报错,会返回undefined

  • delete删除数组元素之后,其他元素的索引值不会改变,数组长度也不会发生变化!

除数组元素之后,其他元素的索引值不会改变,数组长度也*

对象的动态特性

  • 当对象呗创建完成之后,可以随时的为对象新曾成员(属性和方法)
  1. 点语法
    1. 对象名.属性名 = 值;
      2.[ ]关联数组语法

异常

  • 异常最大的特征是会阻断代码的执行
  • try {} catch{}
    • 可以极大提高代码的健壮性
    • 特别消耗资源,不经常使用

手动抛出异常 throw

值类型

  • 直接存储的就是数据本身的这种数据,就是值类型的数据
    • 当值类型的数据进行赋值的时候,会将现有变量存储的数据直接赋值一份交给新的变量,此时两个变量虽然值一样,但是存储的空间完全独立,修改其中一个,另一个不受影响

NULL 和 undefined 都是值类型数据

引用类型

  • 储存的数据的地址.数据单独存储起来的这种数据,就是引用类型的数据
  • 引用类型的数据在赋值的时候,会将当前变量中存储的对象的地址赋值一份交给新的变量,此时两个变量同时指向了同一个对象,修改其中一个,另外一个也会受到影响
