XML,JSON

XML
  • XML,JSON
  • 客户机与服务器概念 ,POST请求,XMLHttpRequest 对象
  • 服务器到浏览器的数据转换:
    • 把php数据或对象转成xml或json格式字符串
    • 反之,把json格式转换成php数组或对象
    • 方便数据的传递
  • 浏览器到服务器的数据转换:
    • 在Js中提供方法,把json字符串,转成数据或对象
    • 方便操作
    • 统一的数据格式.其作用可表示复杂数据,便于在各平台之间传输

XML

规范:

  • 可扩展标记语言

  • 第一行必须是版本信息

  • 必须有一个根元素(有且仅有一个,标签可自定义)

  • 标签不可有空格,不可以数字或 . 开头,大小写敏感
  • 不可交叉嵌套,都是双标签
  • 属性双引号
  • 特殊符号要使用实体
  • 注释和HTML一样

php中读取XML文件的内容

  • file_get_contents()
    • 功能: 读取文件内容
    • 参数: 文件的地址
    • 返回值: 内容字符串
  • 默认返回的是text/html文件形式.若是以xml文本原样形式输出,则添加

html中读取XML文件的内容

  • 接收数据: xhr.responseXML
    • 是以xml文档对象的形式接收后台数据
    • 把数据添加到HTML页面:

JSON

  • 定义与规范
    • 轻量级的文本数据交换格式,独立于语言
    • 键值对使用双引号包裹
    • 最后一项不可以有逗号
    • 花括号保存对象, 方括号保存数组
      • 独立于语言,在各平台均可使用
  • 头部定义为

PHP操作JSON ( json_ )

php数组对象 ====== >> json字符串

  • 转换: json_encode(数组名)
  • 读取: file_get_contents('JSON文件的地址值')
  • 返回的是json字符串

json字符串 ===== >> php数组对象

转换: json_decode()

  • 参数1: 要转成数组,或对象的json字符串
  • 参数2: 指定转换成多维数组的形式
    • true指定转换成多维数组的形式

HTML操作JSON ( JSON. )

js数组,对象 ==== >> json字符串

  • JSON.stringify() 转换

json字符串 ===== >> js数组,对象

JSON.parse() 转换

  • 通过获取数组对象的方式进行操作即可;

HTML接收数据:

  • 返回的是xml
    • 则用xhr.responseXML接收
    • content-type:text/xml;charset=utf8;
  • 返回的是json
